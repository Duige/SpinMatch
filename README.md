## 📂 项目结构

```
spinmatch-github/
├── frontend/
│   ├── pages/          # 6个页面
│   │   ├── index.html      # 首页
│   │   ├── booking.html    # 预订中心
│   │   ├── community.html  # 社区动态
│   │   ├── events.html     # 赛事活动
│   │   ├── news.html       # 新闻资讯
│   │   └── profile.html    # 个人中心
│   ├── styles/         # CSS样式
│   ├── scripts/        # JavaScript
│   └── assets/         # 静态资源
├── vercel.json         # 部署配置
└── package.json        # 项目配置
```

## 🛠 本地开发

```bash
# 克隆项目
git clone https://github.com/yourusername/spinmatch.git
cd spinmatch

# 启动开发服务器
npm start
# 或者
python -m http.server 8000 --directory frontend

# 访问 http://localhost:8000
```

## 💡 特别说明

**关于视频文件：**
- 此版本为 GitHub 优化版，已移除大型视频文件
- 网站功能完整，只是暂时使用静态背景

**技术栈：**

- 部署：Vercel 静态托管
